# 启动故障排查

## 1. 必须先完整解压

不要在 WinRAR、7-Zip 或 Windows 压缩包预览窗口中直接双击 BAT。

## 2. 启动入口

优先双击：

- `启动中文版.bat`

中文文件名异常时使用：

- `START_WINDOWS.bat`

## 3. Python

要求 Python 3.10 或更高版本。安装时勾选：

- `Add Python to PATH`

## 4. 查看日志

启动失败后，打开项目根目录的：

- `启动日志.txt`

窗口也会停留显示错误，不再一闪而过。

## 5. 公司或校园网络

如果依赖安装失败，可能是代理、防火墙或 PyPI 访问受限。可切换网络后重新双击启动器。
